# PCB／控制板 — 客戶端需求確認表單

單一靜態 HTML 表單（**4 題必選 + 備註選填**），**單頁捲動**，部署於 Vercel。

## 頁面

| 路徑 | 說明 |
|------|------|
| `/` | 表單本體 |
| `/form-simple.html`、`/form-full.html` | 舊連結會 **302 轉址** 到 `/` |

## 本機預覽

以瀏覽器開啟根目錄的 `index.html`。

## 部署到 Vercel

1. Import 此 Git 儲存庫。
2. Framework：**Other**；不需 Build；Output 留空。
3. 部署完成後將網址提供給客戶即可。

## 表單結構（摘要）

- **第 1～3 題**：製作範圍、原板提供、板子狀況（選項與先前簡易版相同邏輯）。
- **第 4 題**：現場電壓（下拉選單）。
- **備註**：聯絡人／公司、其他補充（數量、交期、動作描述等建議寫在此）。
- 送出後產生文字，可 **傳送到 LINE** 或 **複製**。
